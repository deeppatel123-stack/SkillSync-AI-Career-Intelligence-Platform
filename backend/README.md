# SkillSync Backend

Express + MongoDB + Mongoose API for the SkillSync MERN project.

## Quick start

```bash
cd backend
npm install
# Start MongoDB locally, then:
npm run seed
npm run dev
```

API base: `http://localhost:5000/api`
